# concept
RISC-V: RISC-V is the basic “language” a computer processor uses to understand instructions.
QEMU: a program that simulates a computer so you can run an operating system or CPU architecture inside your own machine.
xv6: xv6 is a small, simplified Unix-like operating system made for teaching.
GDB: GDB (GNU Debugger) is an indispensable tool. It allows you to pause the kernel or user programs at any instruction, inspect variables, examine CPU registers, and step through code line-by-line.
